# BC DBViewer

BC DBViewer是一个基于PyQt5开发的SQLite数据库查看器，支持数据库连接、表数据查看、编辑和管理等功能。

## 功能特点

- 连接和浏览SQLite数据库文件
- 查看表结构和数据
- 支持编辑表数据（修改、添加、删除）
- 友好的图形用户界面
- 支持样式切换

## 系统要求

- Python 3.7+
- PyQt5
- sqlite3 (Python标准库)

## 安装方法

### 1. 克隆或下载项目

```bash
# 克隆项目（如果使用版本控制）
git clone 

# 或直接下载项目文件到本地目录
```

在项目目录下执行以下命令安装必要的依赖：

```bash

```

## 运行方法

### 直接运行

在项目目录下执行以下命令：

```bash
python main.py
```

## 使用指南

1. **连接数据库**：点击"连接数据库"按钮或从文件菜单中选择"打开数据库"，选择SQLite数据库文件。
2. **选择表**：在表选择下拉框中选择要查看的表。
3. **查看数据**：表数据将显示在主界面的表格中。
4. **编辑数据**：点击"编辑模式"按钮进入编辑模式，双击单元格进行修改。
5. **保存更改**：修改完成后点击"保存更改"按钮保存修改。
6. **添加行**：从编辑菜单中选择"添加行"，输入新行的数据。
7. **删除行**：选择要删除的行，从编辑菜单中选择"删除行"。
